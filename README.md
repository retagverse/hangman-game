#  Hangman Game (C++)

##  Description
This is a simple console-based Hangman game built using C++.  
The player tries to guess a hidden word by entering one letter at a time.

The game includes a scoring system to make it more interactive and fun.


## Features
- Simple text-based gameplay
- Word guessing logic
- Limited number of attempts
- Scoring system (start with 100 points)
- Bonus points for remaining tries
- Input validation for correct/incorrect guesses



##  How to Play
1. Run the program
2. Try to guess the hidden word letter by letter
3. Each wrong guess reduces your tries and score
4. Win by guessing the full word before tries run out
5. Final score is displayed at the end


##  Technologies Used
- C++
- iostream
- string library


## Scoring System
- Start: 100 points  
- Wrong guess: -10 points  
- Win bonus: remaining tries × 20
